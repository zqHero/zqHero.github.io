# 三十岁,一切刚刚开始

《三十岁,一切刚刚开始》原作者：李尚龙

# 一，轨迹

1，真实的你，是否一直在努力？问心无愧？

2，三十岁，真正的人生才开始，一分为二看待别人。跟随主流，保持自己永远年轻。

3，寻找一个伟大的目标，大到足够可以改变世界，用一生去完成！！或者做一件持续升值的事情，直到永远。

4，具备远见！！！

5，即使前路迷茫，也要坚持热爱！！

6，生命无常，谋事成功取决于人生态度，你必须倾力而为，老天才会助力相推。

7，限制别人，是恶。别人的人设设计不了你的人生和未来。努力打破不属于自己的人设。

8，莫让自己看起来很精致。尊重知识的人，也会被知识尊重。

9，贫穷和衰老是你唯一不用努力就能获得到的。

# 二，守则

1，如果一无所有怎么办？！坚守更都重要，即使一无所获。凝聚你所有的注意力，去做成一件事情。

2，锻炼身体，定期体检，开始存钱！！热爱学习，多读书！

3，放弃执念，接受这个世界的不完美。和自己和解，接受自己的不完美，坚持热爱！！

4，你可以放松，但不能放纵！打败焦虑最好的方式，就是努力去解决这些让你焦虑的事情！！！！

5，孤独是一个人的狂欢，相聚是一群人的孤单。孤独是你最好的升值期。

# 三，转折

1，感谢挫折，挫折会让你痛苦，但也会让你更强大。

2，成熟：不因不可控的事情而失态，不过分执念，不把情绪放在脸上。

3，莫让自己只是看起来很努力。。你只有足够拼搏，才能无悔！你若想改变世界，请先改变自己，从身边开始。

4，知识是最好的上升阶梯。做好长期规划，对超出认知的东西不要充满执拗和偏见。

5，三十岁之前，做正确的工作，交往值得交往的人。

6，尊重专业人士，因为我们对专业的力量，一无所知。。

# 四，自控

1，生活的平衡，生活没有一个万能的公式，也没有什么方法和途径拿来即可使用。一边前行，一边调整，在前行中平衡自己。。

2，恐惧是成长最大的敌人。恐惧不会让一个人获得自己想要的一切，消除恐惧你才有更多可能。

3，自律给你更好的生活。

4，痛苦的根源，在于自己。

# 五，故事

略，，，作者的故事。1，我请你，吃顿饭。2，离别是为了更好的相见。3，考砸了，生活依旧。4，中年危机前，规划好你要做的事。。。

后记：

学会反思，这是最好的时代，也是最坏的时代。这个世界需要反思的人，

做到优秀是不够的，你必须做到卓越。

这个世界只有自己可以治愈自己。

人为什么要奋斗，因为你要选择，你要有说不的权利！！

-2020-8-8
